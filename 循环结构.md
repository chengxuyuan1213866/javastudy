# 循环结构

## 1.while 循环 

```java
while(条件判断){
    语句;
}
```

 

## 2.do ~while循环

```java
do{
    语句;
}while(条件判断)
```

- do-while循环是先执行一遍循环体，然后再判断循环条件是否成立，即使循环条件不成立，也会执行一遍语句

## 3.for循环

```java
for(初始化循环变量;判断条件语句;控制条件语句)
{
	循环体语句;    
}

/* 执行流程：

        A:执行初始化语句

          B:执行判断条件语句，看结果是true还是false

             如果是true，就继续执行

             如果是false，就结束循环

       	   C:执行循环体语句

     	   D:执行控制条件语句

 	       E:回到B继续
 */

```
